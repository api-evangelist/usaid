# USAID

U.S. Agency for International Development (USAID) APIs.json profile for api-evangelist.

USAID provides programmatic access to foreign assistance data, development finance records, project tracking, aid flow analysis, and global development datasets through a suite of free, publicly accessible REST APIs.

## APIs

- **ForeignAssistance.gov Data API** - OAS 3.0 API for U.S. foreign assistance budgetary and financial data across all U.S. government agencies. Covers spending, obligations, and disbursements filterable by country, sector, year, and agency. Docs: https://foreignassistance.gov/api-docs
- **USAID Development Data Library (DDL)** - Open datasets via the Socrata Open Data API (SODA) at data.usaid.gov. Supports filtering, querying, and aggregation of USAID development activity datasets. Register for a free application token for higher rate limits.
- **USAID Development Experience Clearinghouse (DEC) API** - Programmatic access to USAID's repository of international development documentation including technical reports, evaluations, and project documents. Docs: https://dec.usaid.gov/api/help.html
